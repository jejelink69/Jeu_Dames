-Afin de pouvoir ouvrir le projet sur netBeans il faut installer le plugin Gradle (outils>plugin>rechercher gradle)
-Une fois le pluggin intallé, le projet peut être ouvert comme tout projet Java sur netBeans.

-Pour lancer l'interface il faut trouver la classe DesktopLauncher, qui se trouve sous desktop>src>com>ptut>dames>desktop
-Une fois le DesktopLaucher trouvé il faut l'éxécuter, comme une classe main.

-L'interface se lance sur l'écran d'acceuil.
-Cliquez sur le bouton Play.
-Un menu s'offre a vous, vous pouvez:
	-jouer une partie contre l'IA
	-jouer une partie contre un joueur
	-quitter le jeu
	-accéder au credits: nom des réalisateurs et du tuteur
	-accéder aux paramètres(bouton engrenage): modification du damier
	-accéder aux règles du jeu(bouton aide "i")

-Une fois que vous avez chosis de jouer (contre IA ou joueur), vous pouvez rentrer les noms des joueurs et modifier
 l'apparence des pions.
-Vous pouvez ensuite lancer la partie.
-L'interface avec le damier et les pions s'affiche, vous pouvez déplacer les pions en cliquant.

NOTE: Le coeur fonctionnel n'a pas été totalement intégr à 'IHM, il n'est donc pas possible de jouer réélement
pour l'instant
